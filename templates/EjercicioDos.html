EJERCICIO DOS

Realiza un programa para un supermercado.
- Si se vende leche de marca "Asturiana", el descuente es de un 5%.
- Si se vende leche de marca "Pascual", el descuento es de un 10%.
- Si se vende otro producto, no se aplica ningún descuento.
El usuario introducirá el artículo que se vende (leche, galletas, pan..) y la marca de dicho artículo. El programa presentará el descuento correspondiente por pantalla.
